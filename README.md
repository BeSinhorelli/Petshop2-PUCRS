# PetShop Amigo Animal - Projeto da Disciplina web

## 📋 Sobre o Projeto
Este é um sistema web desenvolvido em HTML, CSS e JavaScript para um petshop, criado como parte da Fase 2 do Projeto da Disciplina. O sistema apresenta produtos e serviços oferecidos pelo petshop de forma organizada e acessível.

## 🎯 Objetivo do Sistema
Criar uma plataforma online intuitiva para que clientes possam visualizar os produtos e serviços oferecidos pelo petshop, com informações detalhadas sobre cada item.

## 📁 Estrutura do Site
O site é composto por 4 páginas principais:
- **index.html** - Página inicial com produtos em destaque
- **produtos.html** - Catálogo completo de produtos por categoria
- **servicos.html** - Serviços oferecidos com preços e opções
- **contato.html** - Formulário de contato e informações da empresa

## 🛍️ Produtos Disponíveis
### Acessórios
- Roupas e camisetas
- Camas confortáveis
- Brinquedos diversos

### Rações
- Rações para filhotes
- Rações para adultos
- Rações para cães idosos

### Higiene
- Shampoos e produtos de banho
- Tapetes higiênicos
- Fraldas descartáveis

## 🐕 Serviços Oferecidos
- Banho completo
- Banho + Tosa
- Tosa especializada
- Opção de tele-busca disponível

## 💻 Funcionalidades
- Navegação entre páginas via menu superior
- Visualização de produtos com imagens, descrições e preços
- Tabelas organizadas por categorias
- Informações detalhadas sobre serviços
- Formulário de contato
- Rodapé com informações de contato e horários

## 🔧 Como Executar o Projeto
1. Faça o download de todos os arquivos
2. Crie a pasta `imagens` e adicione as imagens necessárias
3. Abra o arquivo `index.html` em qualquer navegador web
4. Navegue pelo site usando os links do menu
